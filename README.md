# Life

Sandbox for building and showcasing resourceful/life-sim systems — environment 
generation, AI, combat, traffic, weather, quests, vehicles, and more. No launch 
goal; a perpetually evolving portfolio of decoupled, event-driven gameplay architecture.

## Why this exists

A living systems lab bult to showcase my skills monitor growth and learn new features or mechanic creations.
Each feature is built to be a standalone, well-documented example of a specific 
gameplay/architecture pattern, wired together through a shared event-driven core 
so the whole thing coheres into a life-sim sandbox without any single system 
depending directly on another.
